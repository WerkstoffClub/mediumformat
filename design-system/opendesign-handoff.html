Handoff & specs

Everything a designer needs to rebuild Medium Format in OpenDesign: tokens, type, components and the screen set. Values mirror tokens.json and DESIGN.md.

Import into OpenDesign — start the daemon (pnpm tools-dev), open the tokens panel and import design-system/tokens.json (W3C DTCG). Every colour, text style, spacing step, radius and shadow arrives as named, themeable styles. Full steps in design-system/IMPORT.md. v2.1 adds the channel colour key (color.channel.*).
