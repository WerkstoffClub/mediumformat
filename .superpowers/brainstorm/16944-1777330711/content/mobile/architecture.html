Request Flow
1. Client apps (Storefront, Backoffice, POS) send requests through Nginx to the NestJS API core engine.
2. The API resolves auth, reads from PostgreSQL or Redis cache, and delegates to specialized services (Sync Engine, Order Processor).
3. Integration layer bridges external platforms — pushing stock to Tokopedia/Shopee, pulling orders, fetching Discogs metadata, and routing payments via Xendit.
